# Minitalk
# This project is a small data exchange between 2 programs using UNIX signals
  see the ``en.subject.pdf``
## Final grade : 125/100

Mandatory part : 100/100

Bonus : 25/25

### How to use it
Using ``make`` will create the ``server`` and ``client`` executable.

Simply run it with :

For server :

```
./server
```
For client :

```
./client <pid> <message>
```
